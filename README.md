# 土地家屋調査士 一問一答○× Webアプリ

Discord Bot（sikenbot）の一問一答学習をWebアプリ化したものです。
サーバー・データベース・ログイン不要の完全静的アプリで、bolt.new にアップロードして
「Publish」ボタンを押すだけで公開できます。

## 機能（Discord Bot から移植）

- **一問一答○×**: 平成27年〜令和7年の過去問 856肢。○ / × ボタンで回答
- **ランダム出題・一巡管理**: 全856問を一巡するまで同じ問題は出ない（一巡後リセット）
- **正誤判定＋肢別AI解説**: 回答すると即座に判定と解説を表示
- **攻略率**: 一度でも正解した問題数 ÷ 856。プログレスバーで常時表示、初正解時は「🆕 新規攻略！」
- **称号**: 見習い → 補助者(10%) → 測量士補(25%) → 測量士(50%) → 調査士(75%) → ベテラン調査士(90%) → 筆界の達人(100%)。マイルストーン到達時は🎉祝福表示
- **ストリーク**: 連続学習日数🔥・連続正解数⚡（最長記録も統計に表示）
- **復習出題（忘却曲線）**: 間違えた問題を 翌日 → 3日後 → 7日後 に自動再出題。7日後にも正解で卒業
- **統計**: 総回答数・全体/直近30日正解率・年度別正解率・間違いワースト10・正解率トップ10
- **年度フィルター**: 特定年度に絞った出題も可能

## データの保存について

成績・攻略率・復習キューはすべて **ブラウザの localStorage** に保存されます。

- サーバーは一切不要（だから無料・高速で公開できる）
- 同じブラウザで開けば続きから学習できる
- 注意: ブラウザの履歴削除でデータも消える／別の端末とは共有されない

## ローカルでの起動

```bash
npm install
npm run dev
```

## bolt.new での公開手順

1. https://bolt.new を開く
2. プロジェクトをインポートする（どちらでも可）
   - このフォルダを ZIP にせずそのままドラッグ＆ドロップ（`node_modules` と `dist` は除いてよい）
   - または GitHub にプッシュして `bolt.new/~/github.com/ユーザー名/リポジトリ名` を開く
3. Bolt 上でプレビューが動くことを確認
4. 右上の **Publish** ボタンを押す → Bolt Cloud Hosting で公開完了

## 問題データの更新

問題データは `src/data/questions.json`（Discord Bot の `marubatsu_questions.json` と同一形式）。
Bot 側で `parse_marubatsu.py` を実行して JSON を再生成したら、このファイルに上書きコピーするだけで反映されます。

```
discord_quiz_bot\marubatsu_questions.json → src\data\questions.json
```
